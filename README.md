# Task To-Do List Application

## Description
This is a console-based Task To-Do List Application developed in C++ using a
Singly Linked List. It allows users to add tasks, delete completed tasks, and
view all pending tasks.

## Data Structure Used
- Singly Linked List

## Features
- Add new task
- Delete task (mark as done)
- View all tasks
- Dynamic memory usage
